# Gemini CLI

## 它是什么

Gemini CLI 是 Google 的终端 AI 助手，适合交互式问答、代码理解和命令辅助。

## 安装

常见安装方式包括：

```bash
npx @google/gemini-cli
npm install -g @google/gemini-cli
brew install gemini-cli
sudo port install gemini-cli
```

## 登录 / 认证

支持几种常见路径：

- Google 登录
- API Key
- Vertex AI 相关配置

如果你是新手，建议先用 Google 登录，最容易跑通。

## 验证

启动：

```bash
cd myprojectfloder
gemini
```

如果可以进入交互界面并正常响应，就说明基本可用。

## 最小可用示例

你可以让它：

- 解释一个文件
- 生成一个命令建议
- 以 JSON 输出一个简单结果
