# Codex

## 它是什么

Codex 是 OpenAI 的终端型编码助手，适合在项目里直接处理代码、测试、修复和任务协作。

## 安装

### Windows

- 用 npm 安装：`npm install -g @openai/codex`
- 或从 [OpenAI Codex Releases](https://github.com/openai/codex/releases/latest) 下载适合 Windows 的官方发布包

### macOS

- 用 Homebrew 安装：`brew install --cask codex`
- 或从 [OpenAI Codex Releases](https://github.com/openai/codex/releases/latest) 下载官方发布包

### Linux

- 用 npm 安装：`npm install -g @openai/codex`
- 或从 [OpenAI Codex Releases](https://github.com/openai/codex/releases/latest) 下载对应平台的官方发布包

- 如果官方当前文档给出了更具体的 Linux 安装方式，优先按官方说明执行

## 登录 / 认证

启动后通常会让你选择：

- 使用 ChatGPT 登录
- 或使用 API Key

新手优先建议：

1. 先用 ChatGPT 登录
2. 如果你明确需要脚本化或自动化，再走 API Key

## 验证

安装完成后运行：

```bash
codex
```

如果你看到交互界面并完成登录，说明安装成功。

也可以试：

```bash
codex app
```
